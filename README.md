# Browser extension build

A loadable Chrome extension lives in **`sales-automation/`**.

## Install (first time)
1. Green **Code** button → **Download ZIP**.
2. Unzip it — inside is `sales-automation/`.
3. Move `sales-automation/` to a **fixed folder** you'll keep (e.g. `~/Desktop/ext/sales-automation`).
4. Chrome → `chrome://extensions` → enable **Developer mode** → **Load unpacked** → pick that folder.

## Update
1. Download the ZIP again.
2. Unzip and **replace** the files in your fixed folder (overwrite).
3. `chrome://extensions` → hit **reload ↻**.
